# The Bench [![Build Status](https://travis-ci.org/mrrusof/the-bench.svg?branch=master)](https://travis-ci.org/mrrusof/the-bench)

## Requirements

- Bash 3.2
- Python 2.7
- Make 3.81
- Docker 17.03

## Build All Judges

```
make
```
## Clean All Judges

```
make clean
```

## Judges

### Ruby

Build Ruby Judge by executing the following command.

```
cd ruby && make
```
